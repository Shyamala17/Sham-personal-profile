# Persistent Dark/Light Theme Toggle

A simple web project demonstrating a **persistent dark/light mode feature** with **clean theme management logic** using CSS variables, JavaScript, and LocalStorage.

- **CSS Variables**: Centralized color definitions for background, text, and accent.
- **Two Theme Configurations**: Light (default) and Dark, managed via `[data-theme]`.

  document.documentElement.setAttribute("data-theme", currentTheme);

document.getElementById("theme-toggle").addEventListener("click", () => {
  let theme = document.documentElement.getAttribute("data-theme");
  let newTheme = theme === "dark" ? "light" : "dark";
  document.documentElement.setAttribute("data-theme", newTheme);
  localStorage.setItem("theme", newTheme);

ğŸ”¹ LocalStorage
Browser storage for key-value pairs (~5MB).

Data persists across sessions until cleared.

Example: localStorage.setItem("theme", "dark");

ğŸ”¹ Why Store Preferences
Improves UX by remembering user settings (theme, language, etc.).

Cons: can cause halation for users with astigmatism.
